# Entropy Ecology Simulator

This is a production-ready snapshot of the `entropy_ecology_sim` project as discussed.

- JAX-native RSVP dynamics core
- Agent types (autocatalytic, pumps, builders, parasites)
- Metrics for power laws, transfer entropy, Fisher information, regime shifts, and Lyapunov proxy
- Minimal experiment runner
- Skeleton models for forest gaps, supply chains, and meme diffusion
- Basic visualization stubs
